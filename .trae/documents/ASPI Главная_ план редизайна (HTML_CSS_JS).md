## Информационная архитектура страницы

* Header: верхняя утилити‑панель (соц.сети, email/телефон) + основной навбар; липкий (sticky); активный пункт — синяя таблетка `#3b5494`.

* Hero‑слайдер: 4–6 слайдов по ключевым направлениям, первый — Green Energy; полупрозрачный тёмный оверлей, заголовок, слоган, кнопка `Learn more`. Навигация — стрелки/точки, анимация fade/slide.

* Trust‑bar: короткая полоса под героем с бейджем FIDIC «Since 1999», KPI (25+ лет, 200+ проектов, 15+ стран).

* About: 2–3 абзаца (“Who We Are / Why ASPI”) + фото/видео; миссия в одном абзаце, ссылка на About.

* Services Overview: сетка из 7 карточек (первой — Green Energy, визуально выделена мягким зелёным фоном). В каждой: иконка, название, 1–2 предложения, `Read more`.

* Featured Projects: 3–6 карточек, кнопка `View all projects`.

* Partners / Certificates: логотипы международных партнёров + бейджи (FIDIC, ISO 9001/14001).

* Contact / CTA: «Let’s work together» + `Get in touch`, адрес/телефон/email; модальная форма.

* Footer: дубликат меню, монохромный логотип, социальные и копирайт.

## Компоненты и стили

* Навбар: адаптивный гамбургер, прозрачность при скролле, контрастные состояния `:hover/:focus`.

* Кнопки: `primary` (градиент #3b5494→#2f416f), `outline` (контур #3b5494), размеры `sm/md/lg`.

* Карточки услуг/проектов: единый шаблон, ховеры с лёгким подъёмом/тенью, зелёные акценты только для Green Energy.

* Бейджи сертификатов: компактные чипы с иконками.

* Модальные окна: плавное появление, доступность по клавиатуре.

## Взаимодействия и анимации

* Герой‑слайдер: авто‑переключение 6–7 сек, пауза по взаимодействию; fade/slide‑in, ограничить на мобильных.

* Hover‑эффекты: аккуратные, без резких масштабирований; курсор/фокус управляемые.
